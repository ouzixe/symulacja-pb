# Symulacja pola bitwy
Programowanie Obiektowe.

Symulacja agentowa ma za zadanie badać zachowanie oddziałów walczących między sobą. 
Oddziały będą poruszać się po dwuwymiarowej przestrzeni (planszy) o zadanej wielkości. Na niektórych polach znajdują się obiekty, które zmieniają wartości militarne oddziału wchodzącego na pole obiektem.
